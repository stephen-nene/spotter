## üöÄ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- pnpm (v7 or higher) /  npm (v7 or higher)
- Git

  - *git-flow*

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tyneitafrica/tender-hub
   cd tender-hub
   ```

2. Install dependencies:
   ```bash
   npm install
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm run dev
   npm run dev
   ```
